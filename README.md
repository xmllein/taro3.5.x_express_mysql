## taro_express_mysql

- 一个基于`taro+express+mysql`的小程序项目
- node 版本：`v16.15.0`
- 后台服务在`server`目录下, 根目录下`yuanfang.sql`是数据库文件

## 主要功能

- [x] 首页
- [x] 选择城市
- [x] 航班列表页
- [x] 航班详情页
- [x] 用户登录（注册）
- [x] 航班订单列表页
